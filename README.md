# Local File Server

LAN 内で使えるブラウザベースのファイルサーバーです。  
Flask + Socket.IO で動作し、ファイルの閲覧・編集・共有・リアルタイム共同編集などを 1 つの Web UI で提供します。

---

## 必要なもの

- Python 3.10 以上
- pip

### パッケージインストール

```bash
pip install flask flask-socketio markdown markupsafe
```

---

## 起動方法

### Windows（bat ファイル）

`server.bat` をダブルクリックするだけで起動し、ブラウザが自動で開きます。

### 手動起動

```bash
python app.py
```

起動後、ブラウザで `http://localhost:5000/` にアクセスしてください。  
同一 LAN 内の他の PC からは `http://<サーバーのIP>:5000/` でアクセスできます。

---

## ディレクトリ構成

```
Local_File_Server/
  app.py            … Flask アプリ本体
  server.bat        … Windows 用起動スクリプト
  static/
    script.js       … フロントエンド JavaScript
  templates/
    index.html      … UI テンプレート (HTML + CSS)
  files/            … ファイル保存先（サーバー起動時に自動作成）
  .state/           … 内部状態（チャット履歴、お気に入り等）
  .history/         … ファイル変更履歴
  .trash/           … ゴミ箱
```

## 機能一覧

### ファイル操作

| 機能 | 説明 |
|------|------|
| ブラウズ | フォルダ階層をたどってファイルを閲覧 |
| アップロード | ドラッグ＆ドロップまたはボタンから（500 MB まで） |
| ダウンロード | ファイル単体 or フォルダを ZIP で一括ダウンロード |
| 新規作成 | フォルダ・空ファイルの作成 |
| リネーム | ファイル名・フォルダ名の変更 |
| コピー | 同一フォルダ内に `ファイル名_copy` としてコピー |
| 移動 | ファイル・フォルダの移動（一括移動対応） |
| 削除 | ゴミ箱に移動（復元可能）。ゴミ箱から完全削除も可能 |
| 一括操作 | チェックボックスで複数選択 → 一括削除・ZIP・移動 |

### プレビュー

| 対応形式 | 内容 |
|----------|------|
| テキスト | コードハイライト付き表示（highlight.js） |
| 画像 | PNG, JPG, GIF, WebP, SVG, BMP をインライン表示 |
| 動画・音声 | MP4, WebM, MP3, WAV 等をブラウザ内再生 |
| PDF | `<iframe>` で埋め込み表示 |
| Office | Excel (.xlsx) → テーブル表示、Word (.docx) → HTML 変換 |
| Markdown | HTML レンダリング表示 |

### 編集

| 機能 | 説明 |
|------|------|
| テキスト編集 | ブラウザ上でテキストファイルを直接編集・保存（Ctrl+S） |
| リアルタイム共同編集 | 複数ユーザーが同時にファイルを編集可能（Socket.IO） |
| カーソル共有 | 他ユーザーのカーソル位置をリアルタイム表示 |
| 編集ロック | 排他制御で同時書き込みの競合を防止 |
| 変更履歴 | 保存ごとにバージョンが記録され、任意のバージョンに復元可能 |
| 画像編集 | 回転・フリップ等の基本的な画像編集 |

### 検索・整理

| 機能 | 説明 |
|------|------|
| ファイル検索 | ファイル名検索（サブフォルダ再帰対応） |
| フィルタ | 表示中のファイル一覧をインクリメンタル絞り込み（Ctrl+F） |
| ソート | 名前・サイズ・更新日 の昇順/降順 |
| 表示切替 | リスト表示 / グリッド表示 |
| お気に入り | IP 単位でファイルをお気に入り登録 |
| 最近のファイル | 直近に更新されたファイルを一覧表示 |

### コラボレーション

| 機能 | 説明 |
|------|------|
| チームチャット | 全体チャット（Socket.IO リアルタイム） |
| ダイレクトメッセージ | ユーザー間の 1 対 1 チャット |
| オンラインユーザー | 現在接続中のユーザーをリアルタイム表示 |
| 共有リンク | 有効期限付き（72 時間）の共有 URL を生成 |
| 共有クリップボード | ユーザー間でテキストを共有 |
| アクティビティログ | アップロード・削除・リネーム等の操作履歴を記録 |

### ツール

| 機能 | 説明 |
|------|------|
| テキスト比較 | 2 つのテキストの diff を表示 |
| ターミナル | ブラウザ上からサーバー側でコマンドを実行 |
| ストレージ情報 | ディスク使用量をヘッダーに表示 |
| ファイル詳細 | サイズ・作成日・MIME タイプ等の詳細情報を表示 |
| パスコピー | ファイルパスをクリップボードにコピー |

### UI

| 機能 | 説明 |
|------|------|
| ダークモード | ライト / ダーク テーマ切替 |
| パネルリサイズ | ファイル一覧とプレビューの幅をドラッグで調整 |
| レスポンシブ | モバイル表示対応 |
| 右クリックメニュー | ファイルの右クリックで操作メニュー表示 |
| キーボードショートカット | Ctrl+S（保存）、Ctrl+F（フィルタ）、Escape（閉じる） |

---

## キーボードショートカット

| キー | 動作 |
|------|------|
| `Ctrl + S` | 編集中のファイルを保存 |
| `Ctrl + F` | フィルタ入力にフォーカス |
| `Escape` | モーダル / コンテキストメニューを閉じる |

## 設定

`app.py` 冒頭の定数で動作を調整できます。

| 定数 | デフォルト | 説明 |
|------|-----------|------|
| `BASE_DIR` | `./files` | ファイル保存ディレクトリ |
| `MAX_CONTENT_LENGTH` | 500 MB | アップロード上限サイズ |
| `CHAT_HISTORY_LIMIT` | 80 | チャット履歴の保持件数 |
| `DM_HISTORY_LIMIT` | 100 | DM 履歴の保持件数 |
| `ACTIVITY_LOG_LIMIT` | 200 | アクティビティログの保持件数 |
| `SHARE_LINK_EXPIRE_HOURS` | 72 | 共有リンクの有効期限（時間） |
| `MAX_TERMINAL_OUTPUT` | 40,000 文字 | ターミナル出力の上限 |

---

## 注意事項

- **ローカルネットワーク向け** です。インターネットに公開しないでください。
- ターミナル機能はサーバー上でコマンドを実行します。信頼できるネットワークでのみ使用してください。
- `files/` ディレクトリが実際のファイル保存先です。バックアップは `files/` を丸ごとコピーしてください。
